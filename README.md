A simple Chrome extension that allows you to copy links from MyBib into Google Docs easily.

# To Install:
* Download the CRX file, then drag it onto the Chrome Extensions page
* Download the source folder, enable developer mode in the Chrome Extensions page, then press the "load unpacked" button and select the source folder
# To Use:
1. Open the MyBib website (MyBib.com)
2. Click the extension icon
3. If the icon changes to a tick, the copy was successful; otherwise, there was an error
4. Paste your clipboard in Google Docs
